# engineering-cookbook 💡

Un livre de recettes. Tu ouvres, tu cherches « **je veux faire X** », tu obtiens une recette complète.

```
Je veux un JWT          → recette
Je veux du Kafka        → recette
Je veux Dockeriser      → recette
Je veux un Circuit Breaker → recette
Je veux un Config Server   → recette
Je veux OpenTelemetry      → recette
```

## Format imposé de chaque recette
- 🎯 Le problème à résoudre
- 🧠 Les concepts nécessaires
- 🏗️ Une architecture minimale
- 💻 Un exemple de code
- ⚠️ Les pièges à éviter
- 🚀 Les variantes selon le contexte

Voir [`_TEMPLATE.md`](_TEMPLATE.md). Créer une recette avec la skill **ajouter-recette-cookbook**.
