# docs/standards

Conventions **transverses** aux projets de `jihedapps`. Uniquement ce qui est spécifique à cet
écosystème — pas les généralités qu'un modèle connaît déjà. À charger à la demande.

| Fichier | Domaine |
|---|---|
| [java-spring.md](java-spring.md) | Backend Java 17 / Spring Boot 3.2.5 |
| [angular.md](angular.md) | Frontend Angular 17 / TypeScript |
| [database.md](database.md) | PostgreSQL / JPA / migrations |
| [security.md](security.md) | Auth, secrets, endpoints |

Un projet peut surcharger une règle dans son propre `CLAUDE.md` — il le documente alors explicitement.
