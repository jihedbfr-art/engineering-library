## 🎯 Philosophy

1. **Practice over theory** — every concept comes with a working example.
2. **Security by default** — DevSecOps and safe practices are not afterthoughts.
3. **Production mindset** — knowledge that ships, not knowledge that only demos.
4. **Timeless first** — foundations that outlast any framework, then the tools of today.

## 🧭 Where to start

- **New to the field?** → [Computer Science](computer-science/) foundations, then a language in [Programming](programming/).
- **Building for the web?** → [Web](web/) + [Backend](backend/) + [Frontend](frontend/).
- **Going to production?** → [DevSecOps](devsecops/) + [Cloud](cloud/) + [Software Engineering](software-engineering/).
- **Into security or AI?** → [Cybersecurity](cybersecurity/) · [AI](ai/), each with a [roadmap](resources/roadmaps/).
