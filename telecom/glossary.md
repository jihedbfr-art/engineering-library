# Telecom Glossary

The acronym jungle, decoded. Telecom has more TLAs (three-letter acronyms) than any other industry — keep this open.

## Generations & radio

| Term | Meaning |
|---|---|
| **GSM** | Global System for Mobile — the 2G standard |
| **UMTS / WCDMA** | 3G standard |
| **LTE** | Long-Term Evolution — 4G |
| **NR** | New Radio — the 5G air interface |
| **RAN** | Radio Access Network — antennas + radios |
| **eNodeB / gNodeB** | The 4G / 5G base station |
| **O-RAN** | Open RAN — open, multi-vendor interfaces |
| **mmWave** | High-frequency 5G spectrum (24 GHz+) |
| **MIMO** | Multiple-Input Multiple-Output antennas (more capacity) |

## Core network

| Term | Meaning |
|---|---|
| **EPC** | Evolved Packet Core — the 4G core |
| **5GC** | 5G Core |
| **SBA** | Service-Based Architecture — 5G's microservices core |
| **MME / AMF** | Mobility management (4G / 5G) |
| **HSS / UDM** | Subscriber database (4G / 5G) |
| **S-GW / P-GW / UPF** | User-data gateways (4G / 4G / 5G) |
| **PCRF / PCF** | Policy & charging control (4G / 5G) |
| **NRF** | Network Repository Function — 5G service discovery |
| **IMS** | IP Multimedia Subsystem — voice/video over IP |
| **CUPS** | Control/User Plane Separation |

## Identifiers

| Term | Meaning |
|---|---|
| **IMSI** | International Mobile Subscriber Identity (the SIM/subscriber) |
| **MSISDN** | The phone number |
| **IMEI** | Device hardware identity |
| **ICCID** | SIM card serial number |
| **MCC / MNC** | Mobile Country / Network Code |
| **SUCI / SUPI** | 5G concealed / permanent subscriber identity |
| **APN** | Access Point Name — the gateway to a data network |

## Protocols & signaling

| Term | Meaning |
|---|---|
| **SS7** | Legacy 2G/3G signaling system |
| **Diameter** | 4G AAA + policy/charging protocol |
| **GTP** | GPRS Tunneling Protocol (user/control data in the core) |
| **SIP** | Session Initiation Protocol — sets up VoIP/VoLTE calls |
| **RTP** | Real-time Transport Protocol — carries the media |
| **VoLTE / VoNR / VoWiFi** | Voice over LTE / 5G NR / Wi-Fi |
| **Ro / Rf** | Diameter online / offline charging interfaces |

## Billing & business

| Term | Meaning |
|---|---|
| **CDR / xDR** | Call / usage Detail Record |
| **OCS** | Online Charging System (real-time/prepaid) |
| **CHF** | Charging Function (5G converged charging) |
| **BSS / OSS** | Business / Operations Support Systems |
| **RA** | Revenue Assurance |
| **ARPU** | Average Revenue Per User — the key business KPI |
| **Churn** | Rate at which customers leave |
| **eTOM** | TM Forum's telecom process framework |

## Devices, IoT & SIM

| Term | Meaning |
|---|---|
| **UE** | User Equipment (your device) |
| **M2M** | Machine-to-Machine |
| **NB-IoT / LTE-M** | Low-power cellular IoT technologies |
| **eSIM / eUICC** | Embedded, reprogrammable SIM |
| **iSIM** | SIM integrated into the main chip |
| **RSP** | Remote SIM Provisioning |
| **LPWAN** | Low-Power Wide-Area Network |

## Networks & business models

| Term | Meaning |
|---|---|
| **MNO** | Mobile Network Operator (owns the network) |
| **MVNO** | Mobile Virtual Network Operator (rents & resells) |
| **MEC** | Multi-access Edge Computing |
| **NFV / SDN** | Network Function Virtualization / Software-Defined Networking |
| **FWA** | Fixed Wireless Access (home broadband over cellular) |
| **QoS** | Quality of Service |
| **SLA** | Service-Level Agreement |

## Bodies & standards

| Term | Meaning |
|---|---|
| **3GPP** | Standardizes 3G/4G/5G |
| **GSMA** | Global mobile operators' association |
| **ITU** | UN telecom agency (spectrum, IMT requirements) |
| **TM Forum** | BSS/OSS standards |
| **CAMARA** | Open-source network-API project (with GSMA Open Gateway) |
