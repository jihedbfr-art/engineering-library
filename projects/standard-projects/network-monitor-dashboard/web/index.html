Manuel utilisateur

Outil de supervision de votre réseau local. Tout fonctionne uniquement pendant que l'application est ouverte — rien ne tourne en arrière-plan une fois fermée.

Les écrans

Vue d'ensembleAppareils en ligne, débit temps réel, graphique de bande passante et infos de votre connexion (IP locale, passerelle, IP publique).
AppareilsTous les appareils du réseau : type (routeur, PC, téléphone…), IP privée, nom d'hôte, marque, MAC, durée de connexion et rôle.
TraficHistorique de la bande passante, totaux et pics entrant / sortant.
DébitTest de vitesse à la demande (téléchargement / envoi / ping / gigue). Clic sur Lancer le test.
Flux & destinationsConnexions sortantes de cette machine, regroupées par site / service (Google, GitHub…) avec le temps passé sur chacun.
AlertesJournal des événements de sécurité, filtrable par niveau, avec origine, explication et conseils de défense.
UrgencesUniquement les menaces critiques (rouge) : attaques entrantes, scans de ports, accès non autorisés.
CarteTopologie du réseau (tous les appareils autour du routeur) + point de sortie Internet géolocalisé.
ParamètresChanger le mot de passe admin.

Bien démarrer

  1. Lancer : double-clic sur start.bat (ou python app.py), puis ouvrir http://127.0.0.1:8787.
  2. Se connecter : admin / admin par défaut.
  3. Sécuriser : changer le mot de passe dans Paramètres dès la première utilisation.
  4. Astuce : lancer en administrateur donne une découverte réseau plus complète.

Comment arrêter l'application

  • Depuis ici : le bouton rouge ci-dessous arrête proprement le serveur.
  • Fenêtre du terminal : appuyer sur Ctrl + C, ou simplement fermer la fenêtre noire.

Ce que l'outil ne fait pas

Il ne capte pas les identifiants, mots de passe ou la navigation des autres appareils : ce serait casser leur chiffrement HTTPS (interception illégale). Le bouton Bloquer ne coupe pas Internet par la force — il vous renvoie vers l'admin de votre routeur pour un blocage propre (filtrage MAC).

A propos

Outil de supervision de votre reseau local, usage personnel. Il mesure la presence des appareils, la bande passante et les destinations de votre trafic. Il ne capte pas les identifiants ni la navigation des autres appareils — cela exigerait de casser leur chiffrement HTTPS, ce qui est hors perimetre.
