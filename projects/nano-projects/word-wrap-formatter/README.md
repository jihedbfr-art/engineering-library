# word-wrap-formatter

Reformate un texte pour qu'aucune ligne ne depasse une largeur donnee, en coupant
uniquement entre les mots (jamais au milieu d'un mot).

Lancer :

```
python main.py
```

Exemple avec une largeur de 15 : `the quick brown fox jumps over the lazy dog` devient
trois lignes de 15 caracteres ou moins chacune.
