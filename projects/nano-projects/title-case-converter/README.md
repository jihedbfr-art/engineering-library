# title-case-converter

Convertit un texte en Title Case, en gardant les petits mots (a, the, of, in...) en
minuscule sauf en premiere/derniere position — la vraie convention typographique anglaise,
pas juste "majuscule a chaque mot".

Lancer :

```
python main.py
```

Exemple : `the lord of the rings` -> `The Lord of the Rings`.
