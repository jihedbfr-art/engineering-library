# weather-dashboard-angular

## Vue d'ensemble
Dashboard météo multi-villes sur 7 jours, données 100% simulées (pas d'API réelle — voir README).
Vanilla JS, pas Angular réel.

## Stack
HTML + CSS + JavaScript vanilla, zéro dépendance, zéro build.

## Commandes
Ouvrir `index.html` dans un navigateur.

## Fichiers clés
- `data.js` — données météo simulées codées en dur (4 villes x 7 jours).
- `app.js` — sélecteur de ville, rendu du dashboard.
- `index.html` / `style.css` — structure et style des cartes.
