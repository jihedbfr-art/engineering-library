# recipe-book-app

## Vue d'ensemble
CRUD de recettes avec recherche par titre/ingrédient, persistance `localStorage`.

## Stack
HTML + CSS + JavaScript vanilla, zéro dépendance, zéro build.

## Commandes
Ouvrir `index.html` dans un navigateur.

## Fichiers clés
- `app.js` — état des recettes (titre, ingrédients[], étapes[]), recherche, rendu, persistance.
- `index.html` — formulaire d'ajout (textarea multi-lignes) + zone de recherche + liste.
