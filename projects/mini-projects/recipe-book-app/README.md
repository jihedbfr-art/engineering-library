# recipe-book-app

## But
CRUD de recettes (titre, ingrédients, étapes) avec recherche par titre ou ingrédient.

## Architecture
Vanilla JS/HTML/CSS + `localStorage`. Ingrédients et étapes saisis en zone de texte (une ligne =
un élément), stockés comme tableaux dans chaque recette.

## Lancer
Ouvrir `index.html` dans un navigateur.
