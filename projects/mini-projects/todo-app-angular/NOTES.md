# todo-app-angular

## Vue d'ensemble
Todo list single-page, vanilla JS (pas d'Angular réel — voir README pour la justification).
Ajout/suppression/toggle de tâches, filtre actif/terminé, persistance `localStorage`.

## Stack
HTML + CSS + JavaScript vanilla, zéro dépendance, zéro build.

## Commandes
Ouvrir `index.html` dans un navigateur. Aucune installation nécessaire.

## Fichiers clés
- `index.html` — structure de la page (formulaire, filtres, liste).
- `app.js` — état des todos, logique CRUD, rendu, persistance `localStorage`.
- `style.css` — style de la carte todo.
