# qr-code-generator-ui

Genere un motif visuel deterministe type QR a partir d'un texte (hash simple -> grille noir/blanc).
Pedagogique uniquement : ce n'est PAS un encodeur QR conforme, mais illustre le principe de
transformer du texte en motif reproductible. Pour un vrai QR, voir `qr-code-generator-cli` (Python)
qui utilise la lib `qrcode`.

## Lancer
Ouvrir `index.html` dans un navigateur.
