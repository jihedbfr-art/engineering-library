# expense-tracker

## Vue d'ensemble
Suivi de dépenses avec total et graphique en barres CSS par catégorie, persistance `localStorage`.

## Stack
HTML + CSS + JavaScript vanilla, zéro dépendance, zéro build.

## Commandes
Ouvrir `index.html` dans un navigateur.

## Fichiers clés
- `app.js` — état des dépenses, agrégation par catégorie, rendu du graphique en barres CSS.
- `index.html` — formulaire d'ajout, zone graphique, liste des dépenses.
- `style.css` — style du graphique en barres (`.bar { height: ...% }`).
