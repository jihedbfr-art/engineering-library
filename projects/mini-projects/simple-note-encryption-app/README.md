# simple-note-encryption-app

Chiffre/dechiffre une note avec AES-256-GCM via la Web Crypto API native du navigateur
(PBKDF2 100k iterations, salt + IV aleatoires) - zero dependance, zero backend.

## Lancer
Ouvrir `index.html` dans un navigateur (necessite un contexte securise : file:// fonctionne
generalement, sinon servir en localhost).
