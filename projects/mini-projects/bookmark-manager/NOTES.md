# bookmark-manager

## Vue d'ensemble
CRUD de favoris avec tags, filtre par tag et recherche, persistance `localStorage`.

## Stack
HTML + CSS + JavaScript vanilla, zéro dépendance, zéro build.

## Commandes
Ouvrir `index.html` dans un navigateur.

## Fichiers clés
- `app.js` — état des favoris (titre, url, tags[]), filtre/recherche, persistance `localStorage`.
- `index.html` — formulaire d'ajout, barre de recherche + sélecteur de tag, liste.
