# bookmark-manager

## But
CRUD de favoris (titre, URL, tags) avec filtre par tag et recherche par titre/URL.

## Architecture
Vanilla JS/HTML/CSS + `localStorage`. Les tags sont saisis en une liste séparée par des virgules et
stockés comme tableau ; le filtre par tag est généré dynamiquement à partir des tags existants.

## Lancer
Ouvrir `index.html` dans un navigateur.
