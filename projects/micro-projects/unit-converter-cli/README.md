# unit-converter-cli

Convertisseur d'unités en ligne de commande : longueur (m/km/mile/ft), poids (kg/lb) et
température (C/F/K), via des sous-commandes argparse.

## Lancer

```bash
python unit_converter.py length 5 km mile
python unit_converter.py weight 70 kg lb
python unit_converter.py temperature 100 C F
```

## Exemple

```bash
$ python unit_converter.py temperature 0 C F
0.0 C = 32.0000 F
```
