# tip-calculator-cli

Calcule le pourboire et le total d'une addition à partir d'un montant et d'un
pourcentage, avec répartition optionnelle entre plusieurs convives.

## Lancer

```bash
python tip_calculator.py 100 -p 20 -n 4
```

## Exemple d'usage

```bash
$ python tip_calculator.py 100 -p 20 -n 4
Addition:        100.00
Pourboire (20.0%): 20.00
Total:           120.00
Par personne (4): 30.00
```
