# sms-pdu-decoder-cli

Decode l'en-tete SMSC d'un PDU SMS brut (format telecom historique utilise entre le mobile et
le centre de messages) - une version pedagogique simplifiee, pas un decodeur PDU complet
(qui gererait aussi l'encodage 7-bit GSM, l'UDH multipart, etc.).

## Lancer

```bash
python pdu_decoder.py 07911326040000F0
```
