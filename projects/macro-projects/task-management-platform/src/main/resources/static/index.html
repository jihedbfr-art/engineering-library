Task Management Platform

API REST de gestion de projets et de taches. Pas d'interface graphique complete dans cette demonstration : utiliser un client HTTP (curl, Postman) avec l'en-tete X-User-Id pour simuler l'utilisateur courant.

  • GET /api/users — liste des utilisateurs de demonstration
  • POST /api/projects — creation d'un projet (ADMIN uniquement)
  • POST /api/tasks — creation d'une tache
  • PUT /api/tasks/{id} — modification (ADMIN, ou MEMBER assigne)
  • DELETE /api/tasks/{id} — suppression (ADMIN uniquement)
  • GET /api/tasks/overdue — taches en retard
  • POST /api/tasks/{taskId}/comments — ajout d'un commentaire

Console H2 disponible sur /h2-console (JDBC URL : voir application.properties).
